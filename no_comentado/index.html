¿Qué es esta página?

Esta página fue creada con el fin de poder configurar un usuario y contraseña para poder subir archivos a través de un servidor FTP.

Existe un servidor web que muestra páginas web. Para poder subir tu propia página web, necesitas acceso al mismo. Actualmente la forma de acceder es a través de un servidor FTP. Para acceder a dicho servidor FTP necesitas un usuario y una contraseña.

Si bien la creación de un usuario y contraseña, así como el directorio raíz (chroot folder) del mismo, se puede hacer manualmente, esta página facilita el proceso.

¿Cómo se hizo esta página?

¡Es más sencilla de lo que parece! Simplemente se utilizó HTML y CSS "Vanilla", es decir, sin ningún tipo de framework o librería. Además, se utilizó JavaScript para poder cambiar el tema de la página. Para la parte de backend, se utilizó PHP para realizar las configuraciones en el servidor.

¿Puedo ver el código fuente?

¡Claro! lo puedes ver en este repositorio de GitHub. Se trata de una versión con comentarios y explicaciones para que puedas entender cómo se hizo.
